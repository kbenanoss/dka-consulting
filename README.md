## 🏢 À Propos

**DKA-Consulting Sarl** est un cabinet spécialisé dans le domaine foncier et immobilier au Togo. Créée en 2023 à Lomé par de jeunes diplômés en droit des affaires et en géométrie-topographie, l'entreprise accompagne ses clients dans toutes les opérations liées aux titres fonciers.

### Informations Légales
- **RCCM** : TG-LFW-01-2023-B 12-01723
- **NIF** : 1001889673
- **CNSS** : 180768
- **Régime fiscal** : Sans TVA

## 🎯 Services

- **Création de Titre Foncier** : Procédure d'immatriculation foncière auprès du guichet unique foncier
- **Mutation de Titre Foncier** : Transfert de propriété d'un bien immobilier
- **Morcellement** : Division d'un terrain immatriculé en plusieurs parcelles
- **Conseil Juridique** : Accompagnement dans les questions juridiques
- **Suivi de Chantiers** : Supervision et gestion de projets de construction

## 🚀 Démarrage

### Prérequis

- Node.js 18+ 
- npm ou yarn

### Installation

```bash
# Cloner le dépôt
git clone https://github.com/dka-consulting/website.git

# Accéder au répertoire
cd dka-consulting

# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000) dans votre navigateur.
